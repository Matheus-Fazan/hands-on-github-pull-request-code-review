# Hands On Code Review

Esse projeto foi feito por alunos do _Instituto J&F_, onde foi desenvolvido um site sobre os lugares que os integrantes do grupo têm interesse de conhecer algum dia.


## Objetivo

Com o objetivo de deixar o processo de aprendizagem sobre versionamento, especificamente focado em **pull request** e **code review**, mais eficiente e consolidado. 

## Tecnologia

As tecnologias utilizadas no projeto foram **HTML** e **CSS**, ferramentas aprendidas em aula.
